# CrudCustomer.net

This is a CRUD in .Net with Docker and postgreSQL

### Prerequisites

[Docker](https://www.docker.com/get-started/)

### Run the application

`git clone` [Repo](https://github.com/MarlaGualdron/CrudCustomers.Api.git)

`cd` CrudCustomers.Api

`docker-compose build`

`docker-compose up`
